  • 数据库相关设置
    打开web/WEB-INF/classes/configs.properties,文件头部就是设置数据库部分，代码如下

jdbc.url=jdbc:jtds:sqlserver://127.0.0.1:1433,IP地址为数据库的存放位置的IP地址，本地直接设置127.0.0.1即可，如果是存放在远程端，则需要输入远程端的地址.:1433为sqlserver默认端口，如果数据库服务端有更改为其他端口，配置档需要做相应的端口修改。
DatabaseName=MyDocking为设置数据库的名称，此设置代表数据库名为MyDocking，如果使用其他的数据库，如数据库名为GB2015，直接替换掉成DatabaseName=GB2015即可。
jdbc.username=sa数据库账号设置,此为默认sa账号，若使用其他账号，同理更改即可。
jdbc.password=DZHpoOq/Wgs=密码设置，此密码DZHpoOq/Wgs=为已经加密之后的密码。在前面设置若已关闭加密存储，此处可直接输入明文密码，比如密码是123456,则直接设置成jdbc.password=123456即可.
